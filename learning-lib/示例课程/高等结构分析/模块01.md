# 模块01：矩阵位移法基础

> 用矩阵语言描述结构的受力与变形，是结构数值分析的基石

---

## 核心概念

### 1. 矩阵位移法的基本思想

**定义**：以节点位移为基本未知量，通过单元刚度矩阵组装整体刚度方程，求解节点位移后再回求内力。

**记忆技巧**："先分后合"——先写单元刚度，再组装整体，最后求解。

**与力法的区别**：
| 对比项 | 矩阵位移法 | 力法 |
|--------|-----------|------|
| 基本未知量 | 节点位移 | 多余力 |
| 适用性 | 通用（静定+超静定） | 适合低次超静定 |
| 计算量 | 适合计算机 | 适合手算 |

### 2. 单元刚度矩阵

**定义**：描述单元节点力与节点位移之间关系的矩阵。

**关键公式**：
```
[k]{q} = {F}
```
其中 [k] 为单元刚度矩阵，{q} 为节点位移向量，{F} 为节点力向量。

**记忆技巧**：k 是 stiffness（刚度），q 是 displacement（位移），F 是 Force（力）。

### 3. 坐标变换

**定义**：将局部坐标系下的单元刚度矩阵转换到整体坐标系。

**关键公式**：
```
[k]^g = [T]^T [k]^l [T]
```
其中 [T] 为坐标变换矩阵。

**易混淆**：[T]^T 在等号左边，不是右边。记忆："整体 = 转置 × 局部 × 变换"。
