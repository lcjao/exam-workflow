# 模块01：论文结构与逻辑

> 一篇好论文 = 好故事 + 好逻辑 + 好表达

---

## 核心概念

### 1. IMRAD 结构

**定义**：科技论文的标准结构——Introduction（引言）、Methods（方法）、Results（结果）、And（与）、Discussion（讨论）。

**记忆技巧**："我妈妈爱跳舞" → I-M-R-A-D

**各部分职责**：
| 部分 | 回答的问题 | 核心内容 |
|------|-----------|----------|
| Introduction | 为什么做？ | 背景、问题、目标 |
| Methods | 怎么做？ | 方法、材料、步骤 |
| Results | 发现了什么？ | 数据、图表、发现 |
| Discussion | 意味着什么？ | 解释、对比、意义 |

### 2. 漏斗式引言

**定义**：从大背景逐步聚焦到具体研究问题的写作方式。

**结构**：
```
大背景（领域重要性）
    ↓
小背景（具体问题）
    ↓
知识缺口（还没解决什么）
    ↓
本研究（我们做了什么）
```

**记忆技巧**："从宽到窄，从远到近"。

### 3. 逻辑链条

**定义**：论文中每个论点之间的因果/推理关系。

**常见逻辑错误**：
- 跳跃推理：A → C（缺少 B 的中间步骤）
- 因果倒置：把结果当原因
- 循环论证：用结论证明前提
