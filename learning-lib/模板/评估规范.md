# 评估规范 v2.0

> AI 必须遵循此规范进行评分，确保一致性和公平性

---

## 评分公式

```
模块掌握度 = Σ(题目得分 × 题目权重) / Σ(题目满分 × 题目权重) × 100%
```

## 题型权重

| 题型 | 单题满分 | 权重 | 说明 |
|------|----------|------|------|
| 判断题 | 10 分 | 0.5 | 基础认知检测 |
| 选择题 | 10 分 | 0.7 | 理解程度检测 |
| 简答题 | 20 分 | 1.0 | 核心知识点 |
| 分析题 | 30 分 | 1.5 | 应用能力 |
| 论述题 | 40 分 | 2.0 | 综合能力 |

## 掌握度等级

| 等级 | 分数范围 | 含义 | 行动 |
|------|----------|------|------|
| ✅ 掌握 | 80-100% | 可以进入下一模块 | 推进 |
| ⚠️ 模糊 | 60-79% | 部分理解但有漏洞 | 薄弱点补讲 → 二次评估 |
| ❌ 未掌握 | 0-59% | 理解不足 | 薄弱点补讲 → 二次评估 → 无论如何推进 |

## 评分原则

1. **先肯定后指出问题**
2. **具体到知识点/题号**，不笼统说「需要加强」
3. **给改进方向**，不只是说「答错了」
4. **评分尺度一致**：同类题型用相同标准
5. **量化优先**：尽量用分数而非主观判断

## 各题型评分细则

### 判断题（10分）
- 正确：10 分
- 错误：0 分
- 不适用部分分

### 选择题（10分）
- 正确：10 分
- 错误：0 分
- 多选漏选：酌情给 3-5 分

### 简答题（20分）
| 得分 | 标准 |
|------|------|
| 18-20 | 完整准确，有关键词有展开 |
| 14-17 | 方向正确，但缺少部分要点 |
| 10-13 | 有一个要点正确，但表述不完整 |
| 5-9 | 方向对但严重不完整 |
| 0-4 | 方向错误或未作答 |

### 分析题（30分）
| 得分 | 标准 |
|------|------|
| 27-30 | 分析全面，有数据/案例支撑 |
| 21-26 | 分析方向正确，但缺少深度 |
| 15-20 | 有分析但逻辑不清晰 |
| 9-14 | 方向对但分析浅尝辄止 |
| 0-8 | 方向错误或未作答 |

### 论述题（40分）
| 得分 | 标准 |
|------|------|
| 36-40 | 结构完整（总起-分析-总结），论点清晰，有独立见解 |
| 28-35 | 结构较完整，论点清晰但缺少独立见解 |
| 20-27 | 有论点但结构不完整 |
| 12-19 | 只写关键词，未展开成完整句子 |
| 0-11 | 方向错误或未作答 |

## 特殊处理

| 情况 | 处理方式 |
|------|----------|
| 用户未回答某题 | 记 0 分，列入薄弱点，不强迫 |
| 方向对但表述不完整 | 给部分分（50-70%），提示补充什么 |
| 完全正确但有更好表述 | 给满分，补充进阶说法 |
| 用户说"这个我会了" | 标记为已掌握，跳过该题评分 |
| 用户说"跳过这题" | 标记为跳过，不纳入评分 |
| 用户认为评分有误（申诉）| 重新评估并说明理由 |

## 评估输出格式

```
📊 模块XX 评估结果

总体掌握度：XX%（等级：✅/⚠️/❌）

逐题评分：
  Q1 [简答题] 16/20 权重1.0 → 得分率80%
    ✅ 核心概念正确
    ⚠️ 缺少具体数值支撑，建议补充数据

  Q2 [分析题] 18/30 权重1.5 → 得分率60%
    ❌ 分析方向偏差：混淆了X和Y的概念
    📌 改进：重新阅读模块XX"X与Y的区别"

加权总分：XX分 / 满分XX分

✅ 掌握较好：[具体知识点]
⚠️ 需要强化：[具体问题] → [改进方向]

📌 下一步：[明确可执行的指引]
```
