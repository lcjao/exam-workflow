# ⌨️ 命令速查卡

> 复制命令直接对 AI 说即可

---

## 核心命令

| 命令 | 触发动作 | 使用场景 |
|------|----------|----------|
| **「我答完了」** | AI 读取答案 → 量化评估 → 更新进度 | 答完模块自测题后 |
| **「我学完了」** | AI 评估掌握度 → 决定推进/补强 | 学完但没答题时 |
| **「补答完了」** | AI 二次评估 → 无论如何推进 | 完成薄弱点补强后 |
| **「开始模块XX」** | 生成指定模块的学习卡片 | 开始学习或跳级 |
| **「我要毕业」** | 生成毕业评估 → 判定是否达标 | 所有模块学完后 |

## 辅助命令

| 命令 | 触发动作 | 使用场景 |
|------|----------|----------|
| **「跳过这题」** | 标记该题为跳过，不纳入评分 | 已掌握或不想答 |
| **「这个我会了」** | 标记该知识点为已掌握 | 避免重复学习 |
| **「出综合题」** | 生成跨模块综合练习 | 检验知识串联 |
| **「申诉」** | AI 重新评估指定题目 | 认为评分有误 |
| **「复习」** | 根据复习调度出复习题 | 手动触发复习 |
| **「继续」** | 从上次断点恢复 | 跨会话恢复 |

## 快捷用法

### 开始新课程
```
使用 exam-workflow skill，帮我建立 [科目名称] 的交互备考系统
```

### 指定资料路径
```
我答完了，文件在 D:\...\模块01.md
```

### 跳级
```
开始模块05
```

### 申诉评分
```
申诉，模块03的Q2我认为应该得满分，因为...
```

## 文件权限速查

| 区域标记 | 谁能改 | 内容 |
|----------|--------|------|
| `<!-- AI_READ_ONLY -->` | 只有用户 | 用户答案、自由区 |
| `<!-- AI_WRITABLE -->` | AI 可写 | 评估结果、进度数据 |
| `<!-- USER_ONLY -->` | 只有用户 | 学习笔记、个人感悟 |
